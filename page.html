Project Valhalla Early-Access Builds

This is an early access build from Project Valhalla implementing Value Classes and Objects. See the release notes for details.

The goal of this release is to establish a milestone and solicit feedback as we work toward integrating this functionality into the JDK. Binaries are provided as a convenience so that users do not need to build from the source code.

Warning: This build is based on an incomplete version of JDK 23.
